Capitolo 1: Configurazione & “Ciao, mondo!”

Prepara l’ambiente, scrivi il tuo primo script Python ed eseguilo da console.

Scarica chapter1.py

Obiettivi

  • Installare Python 3.x e verificarne la versione
  • Creare e attivare un ambiente virtuale
  • Scrivere un semplice script “Ciao, mondo!”
  • Eseguire un file .py dal terminale

1. Installazione di Python & configurazione di venv

Vai su python.org/downloads e installa l’ultima release di Python 3. Poi verifica nel terminale: 

python3 --version

Crea una cartella di progetto e spostati al suo interno:

mkdir zero_to_hero && cd zero_to_hero

Inizializza un ambiente virtuale:

python3 -m venv venv

Attivalo:


# macOS / Linux
source venv/bin/activate

# Windows PowerShell
venv\Scripts\Activate.ps1

2. Scrivere “Ciao, mondo!”

Crea src/chapter1.py con:

print("Hello, World!")

Salva ed esegui:

python src/chapter1.py

Dovresti vedere:

Hello, World!

Esercizi

  1. Modifica src/chapter1.py per stampare il tuo nome e l’anno corrente.
  2. Scrivi due istruzioni print() su righe separate.
  3. Disattiva e riattiva il tuo ambiente virtuale; conferma di essere nel venv.
